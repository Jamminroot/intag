# Privacy Policy for InTag

Last updated: January 2026

## Overview

InTag is a Windows utility for managing file and folder metadata (tags, titles, authors, etc.) in Windows Explorer.

## Data Collection

**InTag does not collect, store, or transmit any personal data.**

- All file metadata modifications are performed locally on your device
- No data is sent to external servers
- No analytics or telemetry is collected
- No user accounts or registration required

## Data Storage

InTag modifies Windows file system metadata using standard Windows Shell APIs. This includes:
- NTFS Alternate Data Streams for file properties
- Desktop.ini files for folder tags

All data remains on your local file system under your control.

## Third-Party Services

InTag does not integrate with any third-party services or APIs.
